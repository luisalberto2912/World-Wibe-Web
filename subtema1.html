QUE ES LA WORLD WIBE WEB

Es un sistema de distribución de documentos
de hipertexto o hipermedias interconectados y accesibles vía Internet.
Con un navegador web, un usuario visualiza sitios web compuestos de páginas web que pueden contener textos, imágenes,
vídeos u otros contenidos multimedia, y navega a través de esas páginas usando hiperenlaces.
